# 07函数的扩展和箭头函数
---
### 函数的扩展
```
 //es5 的加法函数
    function add(a,b){
        return a+b;
    }
    console.log(add(1,2));//3
    console.log(add.length);//2 es6提供的 可以获得函数所需参数的个数
    //es6
    function add2(a,b=1){ //通过解构赋值 设置参数b的默认值
        return a+b;
    }
    console.log(add2(1));//2
    console.log(add2.length);//1 原因是只会返回必须输入的参数的个数 不包含有默认值的
```
